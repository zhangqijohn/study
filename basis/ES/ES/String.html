字符串方法
更多方法实例可以参见：JavaScript String 对象。

方法	            描述
charAt()	        返回指定索引位置的字符                                  // "HELLO WORLD".charAt(2) // L
charCodeAt()	    返回指定索引位置字符的 Unicode 值                       // "HELLO WORLD".charAt(2) // 76
concat()	        连接两个或多个字符串，返回连接后的字符串                // "HELLO".concat("WORLD") // HELLO WORLD
fromCharCode()	    将 Unicode 转换为字符串                                 //  String.fromCharCode(76)) // L
indexOf()	        返回字符串中检索指定字符第一次出现的位置                // "HELLO WORLD".indexOf("L") // 2 //语法： string.indexOf(searchvalue,start)
lastIndexOf()	    返回字符串中检索指定字符最后一次出现的位置              //  "HELLO WORLD".lastIndexOf("L") // 9
localeCompare()	    用本地特定的顺序来比较两个字符串                        // ['白鸽', '麻雀', '大象', '狗', '猫', "鸡"].sort((item1, item2) => item1.localeCompare(item2)) // ["白鸽", "大象", "狗", "鸡", "麻雀", "猫"]
match()	            找到一个或多个正则表达式的匹配                          //  "HELLO WORLD".match(/L/g) // ["L", "L", "L"]
replace()	        替换与正则表达式匹配的子串                               // "HELLO WORLD".replace("ELLO","ello") // Hello WORLD
search()	        检索与正则表达式相匹配的值                               // "HELLO WORLD".search("L") // 2;未找到返回为-1
slice()	            提取字符串的片断，并在新的字符串中返回被提取的部分       // "HELLO WORLD".slice(1,5)     // ELLO
split()	            把字符串分割为子字符串数组                               // "How are you doing today?".split(" ",3)   // ["How", "are", "you"]
substr()	        从起始索引号提取字符串中指定数目的字符                   // "HELLO WORLD".substr(1,3) //  ELL
substring()	        提取字符串中两个指定的索引号之间的字符                   // "HELLO WORLD".substring(1,3) //  EL
toLocaleLowerCase()	根据主机的语言环境把字符串转换为小写，只有几种语言（如土耳其语）具有地方特有的大小写映射 // "HELLO WORLD".toLocaleLowerCase() // hello world
toLocaleUpperCase()	根据主机的语言环境把字符串转换为大写，只有几种语言（如土耳其语）具有地方特有的大小写映射 // "hello world".toLocaleUpperCase() // HELLO WORLD
toLowerCase()	    把字符串转换为小写                                       // "HELLO WORLD".toLowerCase() // hello world
toString()	        返回字符串对象值                                         // num.toString(2)   // 1010
toUpperCase()	    把字符串转换为大写                                       // "hello world".toLocaleUpperCase() // HELLO WORLD
trim()	            移除字符串首尾空白                                       //  " HELLO WORLD ".trim() // "HELLO WORLD"
valueOf()	        返回某个字符串对象的原始值

ES6 新增
fromCodePoint()     将 Unicode 转换为字符串;可以识别大于0xFFFF的字符，弥补了String.fromCharCode()方法的不足
raw()               返回一个斜杠都被转义（即斜杠前面再加一个斜杠）的字符串   // String.raw`Hi\\n` === "Hi\\\\n" // true
normalize()         返回给定字符串的Unicode规范化表单（如果该值不是字符串，则它将首先转换为一个字符串）
includes()          返回Boolean;用于判断字符串是否包含指定的子字符串。// str.includes("world",2); // true
startsWith()        用来判断当前字符串是否以另外一个给定的子字符串开头，并根据判断结果返回 true 或 false。  // "hello world".startsWith('hello') // true
endsWith()          用来判断当前字符串是否以另外一个给定的子字符串结尾，并根据判断结果返回 true 或 false。  // "hello world".startsWith('orld') // true
repeat()            构造并返回一个新字符串，该字符串包含被连接在一起的指定数量的字符串的副本。// "Runoob".repeat(2); // RunoobRunoob
padStart()          用另一个字符串填充当前字符串(重复，如果需要的话)，以便产生的字符串达到给定的长度。填充从当前字符串的开始(左侧)应用的。   // 'abc'.padStart(10, "foo");   // "foofoofabc"
padEnd()            用一个字符串填充当前字符串（如果需要的话则重复填充），返回填充后达到指定长度的字符串。从当前字符串的末尾（右侧）开始填充。 // 'abc'.padEnd(10, "foo");   // "abcfoofoof"
trimStart()         移除字符串开始空白                                       //  " HELLO WORLD ".trim() // "HELLO WORLD "
trimEnd()           移除字符串末尾空白                                       //  " HELLO WORLD ".trim() // " HELLO WORLD"
matchAll()          返回一个包含所有匹配正则表达式及分组捕获结果的迭代器。
