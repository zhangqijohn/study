React 全家桶：redux,redux-actions,redux-sagas,immutable.js,react-router ant pro要掌握很多知识点react全家桶 dva umi node ts， 

一 、React: 用于构建用户界面的JavaScript 库
    1.核心： React.Component、 JSX、 Props & State 、 组件API、组件类型

二 、redux: JavaScript 状态容器， 提供可预测化的状态管理。

三 、umi:是一个可插拔的企业级 react 应用框架。umi 以路由为基础的，支持类 next.js 的约定式路由，以及各种进阶的路由功能，并以此进行功能扩展

四 、dva：首先是一个基于 redux 和 redux-saga 的数据流方案，然后为了简化开发体验，dva 还额外内置了 react-router 和 fetch，所以也可以理解为一个轻量级的应用框架。
