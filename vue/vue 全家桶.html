vue全家桶：vue-cli + vue + vuex + vue-router + axios + element-ui

1.vue: 构建数据驱动的web界面前端框架
2.vue-cli：是vue官方提供的快速搭建项目的工具
    npm install -g vue-cli
    vue init  
    vue init webpack my-project
3.vuex:是vue提供的状态管理工具
    “单向数据流”理念的简单示意：    view->actions->state->view
    vuex 核心概念:State 、Getter、 Mutation、 Action、 Module
    Vue Components (Dispatch)--> Actions (Commit) --> Mutations (Mutate) --> State (Render)-->  Vue Components
4.vue-router:是vue生态里面的路由工具，采用路由能轻松实现单页面程序
5.element-ui:一套为开发者、设计师和产品经理准备的基于Vue 2.0 的桌面端组件库.
